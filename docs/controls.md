# Controls

| key(s)        | behavior                     |
| :------------ | :--------------------------- |
| `ctrl+c`, `q` | Quit                         |
| `R`           | Reload sequence file         |
| `h`, `left`   | Move left                    |
| `l`, `right`  | Move right                   |
| `k`, `up`     | Move up                      |
| `j`, `j`      | Move down                    |
| `0`           | Move to first in current row |
| `$`           | Move to last in current row  |
| `g`           | Move to top row              |
| `G`           | Move to bottom row           |
| ` ` (space)   | play/stop toggle             |
